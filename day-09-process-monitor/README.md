# Day 09 - Process Monitoring

## Objective
Analyze system processes to identify top resource consumers.

## Features
- Displays top 5 CPU-consuming processes
- Displays top 5 memory-consuming processes
- Structured report formatting

## Skills Practiced
- Using ps with custom columns
- Sorting output
- Limiting results with head
- Performance analysis basics

## How to Run
chmod +x process_monitor.sh
./process_monitor.sh
