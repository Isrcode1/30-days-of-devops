
# Day 10 - Logging & Report Persistence

## Objective
Create a monitoring script that saves reports to log files.

## Features
- Automatic log directory creation
- Timestamped log files
- Combined terminal + file output using tee

## Skills Practiced
- File handling in Bash
- Using date formatting
- Using tee command
- Directory creation with mkdir -p

## How to Run
chmod +x logging_monitor.sh
./logging_monitor.sh
